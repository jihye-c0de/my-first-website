== 나만의 커뮤니티 기획서 ==

커뮤니티 주제
- 소설/웹툰 정보 공유

디자인 스타일
- 사이트 이름: 최애를 소개합니다
- 색상 테마: 아이보리
- 분위기: 따뜻하고 친근한 분위기
- 반응형 디자인: PC / 모바일 모두 지원 (모바일에서는 왼쪽 사이드바 카테고리를 하단 탭 또는 슬라이드 메뉴로 전환)

특별한 기능 아이디어
- 좋아요, 소설/웹툰 별점, 사진 업로드, 키워드 작성(자유롭게 작성)
- 인덱스 (책갈피 후 나만의 메모 가능한 (소설/웹툰 회차 단위))
- 나의 책장 (독후감, 쓰레기통(읽다가 그만둔 작품 모아두기))

접근 권한 정책
- 비회원(게스트): 게시물 목록/상세, 댓글 등 전체 콘텐츠 열람(보기)만 가능
- 비회원 제한: 별점 등록, 댓글/답글 작성, 좋아요는 회원만 가능 (비회원이 시도 시 로그인 페이지로 이동 안내)

저작권 정책
- 소설/웹툰 표지 이미지: 사용자가 임의로 업로드하지 않고, 관리자가 공식 표지 이미지를 등록/관리
- 사용자 업로드 사진: 독후감·리뷰용 개인 사진(직접 그린 팬아트, 메모 캡처 등)으로 범위 제한
- 게시글/댓글 내 타 사이트 원문 대량 복붙 금지 (신고 대상)

페이지별 특징
- 로그인 페이지
    - 이메일 입력 필드
    - 패스워드 입력 필드
    - 로그인 버튼
    - 비밀번호 찾기(재설정) 링크

- 회원가입 페이지
    - 이메일 입력 필드
    - 이메일 중복 검사 버튼
    - 비밀번호 입력 필드
    - 비밀번호 확인 입력 필드
    - 닉네임
    - 회원가입 버튼

- 게시물 목록
    - 게시물 정보
        - 게시글 제목
        - 평균 별점 (모든 사용자 별점을 합산 후 평균으로 표시, 0.5점 단위 반영)
        - 키워드 (최대 5개)
    - 최신순 / 인기순 / 별점 높은 순 정렬
    - 페이지네이션 또는 무한스크롤
    - 왼쪽 사이드바에 카테고리 (모바일에서는 하단 탭/슬라이드 메뉴로 전환)
        - 소설
            - 장르
            - 키워드
            - 완결 여부
            - 사이트별 정리 (원문 링크 걸어두기)
        - 웹툰
            - 장르
            - 키워드
            - 완결 여부
            - 사이트별 정리 (원문 링크 걸어두기)

- 게시물 상세
    - 게시글 제목
    - 별점
        - 0.5점 단위로 등록 가능 (회원만)
        - 전체 평점은 모든 사용자 별점 합산 후 평균으로 표시
        - 한줄평
    - 키워드
        - 최대 5개 고정
    - 게시글 내용
        - 이미지 (관리자 등록 공식 표지 + 사용자 리뷰 사진 구분 표시)
    - 게시글 작성/수정/삭제 (작성자 본인만 수정·삭제 버튼 노출)
    - 신고 버튼 (부적절한 게시물 신고)
    - 댓글
        - 댓글 작성자
        - 댓글 좋아요
        - 베스트 댓글 상단 고정
    - 비회원은 열람만 가능, 별점/좋아요/댓글 작성 시 로그인 페이지로 안내

- 검색 기능
    - 검색 필드
    - 장르별 검색
    - 키워드 선택
    - 사이트 선택

- 댓글 기능
    - 좋아요 (회원만)
    - 답글 (회원만)
    - 스포 여부 버튼
    - 키워드 최대 5개 등록 가능
    - 댓글 수정/삭제 (작성자 본인만)
    - 댓글 신고 버튼

- 마이페이지
    - 작성한 게시글 보기 (수정/삭제 가능)
    - 댓글을 게시한 글 보기
    - 저장한 게시글 보기
    - 저장한 소설/웹툰
    - 책갈피한 부분 보기
    - 나의 책장(독후감 / 쓰레기통) 관리
    - 알림함 (내 글/댓글에 달린 답글, 좋아요 알림)
    - 회원정보 수정
        - 닉네임
        - 이메일
        - 비밀번호 변경
    - 회원 탈퇴

- 관리자 페이지
    - 관리자 전용 로그인 (일반 회원 권한과 분리)
    - 소설/웹툰 공식 표지 이미지 등록/수정 (저작권 정책에 따른 관리)
    - 신고 접수 목록 조회 (게시글/댓글 신고)
    - 신고된 게시글/댓글 처리 (삭제, 경고, 반려)
    - 회원 관리 (제재, 강제 탈퇴)
    - 카테고리/장르 관리

참고 사이트
- 교보문고 전자도서관, 카카오페이지, 네이버 시리즈
