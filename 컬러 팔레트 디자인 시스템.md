## CSS 변수 정의
```css
:root {
  /* Primary Colors */
  --color-primary: #00D2C5;
  --color-primary-light: #5FE8DC;
  --color-primary-dark: #00A89D;

  /* Secondary Colors */
  --color-secondary: #1B2E5C;
  --color-accent: #FF8FB1;

  /* Background Colors */
  --color-bg-primary: #FFFFFF;
  --color-bg-secondary: #F5F7FA;

  /* Text Colors */
  --color-text-primary: #222222;
  --color-text-secondary: #767676;
  --color-text-muted: #AAAAAA;

  /* Interactive Colors */
  --color-button-primary: #00D2C5;
  --color-button-hover: #00A89D;
  --color-link: #1B2E5C;
  --color-link-hover: #FF8FB1;

  /* Mint Accent (헤더/히어로 등 섹션 배경용) */
  --color-mint: #E3FBF8;
}
```

## 컬러 사용 가이드

### Primary (`--color-primary` / `#00D2C5`)
- 로고 포인트, 주요 CTA 버튼, 뱃지(할인율·태그) 등 브랜드를 대표하는 곳에 사용
- `--color-primary-light`는 hover 배경, 카드 하이라이트, 섹션 배경 틴트 등 넓은 면적에 옅게 사용
- `--color-primary-dark`는 버튼 클릭(active) 상태나 텍스트 위 강조에 사용해 대비 확보

### Secondary (`--color-secondary` / `#1B2E5C`)
- 헤더 로고 텍스트, 메인 타이틀, 히어로 섹션 등 신뢰감이 필요한 영역에 사용
- 어두운 색이므로 본문 텍스트보다는 헤딩·강조 문구 위주로 제한적으로 사용

### Accent (`--color-accent` / `#FF8FB1`)
- 이벤트 배너, 프로모션 문구, 강조 아이콘 등 시선을 끌어야 하는 포인트에만 소량 사용
- 남용 시 브랜드 톤이 흐트러지므로 화면당 1~2곳 이내로 제한 권장

### Background
- `--color-bg-primary`(흰색)를 기본 배경으로, `--color-bg-secondary`(연회색)는 섹션 구분·카드 배경에 사용해 리듬감 부여

### Text
- `--color-text-primary`는 본문/제목, `--color-text-secondary`는 설명·캡션, `--color-text-muted`는 placeholder·비활성 텍스트 등 가장 낮은 우선순위 정보에 사용

### Interactive
- 버튼: 기본 `--color-button-primary` → hover 시 `--color-button-hover`로 톤다운하여 클릭 피드백 제공
- 링크: 기본 `--color-link`(네이비) → hover 시 `--color-link-hover`(코럴)로 전환해 상호작용을 명확히 표시

### Mint Accent (`--color-mint` / `#E3FBF8`)
- 헤더(Navbar), Hero 섹션 등 페이지 상단부의 배경색으로 사용해 다른 섹션과 은은하게 구분
- 그라데이션 대신 단색으로 사용해 톤을 차분하게 유지

## 반응형 고려사항

### 다크모드
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #121212;
    --color-bg-secondary: #1E1E1E;
    --color-text-primary: #F5F5F5;
    --color-text-secondary: #B0B0B0;
    --color-text-muted: #777777;
    --color-primary: #2EE6D9;
    --color-secondary: #7C93C9;
    --color-mint: #16302E;
  }
}
```
- 다크모드에서는 Primary 컬러의 채도를 살짝 높여(`#2EE6D9`) 어두운 배경 위에서도 시인성을 확보
- Secondary(네이비)는 어두운 배경에서 대비가 사라지므로 밝은 톤(`#7C93C9`)으로 대체
- Accent 핑크는 다크모드에서도 원색 유지 가능하나, 배경과의 명도 대비를 반드시 확인

### 다양한 화면 크기
- 모바일 소형 화면에서는 Accent 컬러 사용 영역을 줄여(배너 1개 이하) 시각적 피로도 완화
- 버튼 hover 상태는 터치 디바이스에서 의미가 없으므로, 모바일에서는 `:active` 상태에 `--color-button-hover`를 적용
- 색약 사용자 접근성을 위해 Primary/Accent 색상만으로 정보를 구분하지 않고, 아이콘·텍스트 라벨을 함께 제공
