== Projects 탭 UI/UX 기획서 ==

선택한 레이아웃 방식
[v] 카드 그리드 레이아웃 (2열/3열)

썸네일 카드 구성
이미지: 
- 타입: [v] 스크린샷 [ ] 모바일 목업 [ ] 커스텀 디자인
- 비율: [ ] 16:9 [ ] 4:3 [v] 1:1
- 크기: 가로 ___px, 세로 ___px

정보 표시:
- 프로젝트 이름 (제목)
- 한 줄 설명: _______________
- 기술 스택: [] 뱃지형 [v] 텍스트형 [ ] 아이콘형
- 기타 정보: (개인/팀)

링크 및 버튼:
- [v] Live Demo 버튼
- [v] GitHub 버튼
- [ ] View Details 버튼
- [ ] 기타: _______________

인터랙션 효과
- 호버 효과: [v] 확대 [ ] 그림자 [ ] 이동 [ ] 회전
- 클릭 피드백: [v] 버튼 애니메이션 [ ] 로딩 표시
- 모바일 터치: [v] 터치 피드백 [ ] 스와이프 지원

반응형 설계
- 데스크톱: 3열 그리드
- 태블릿: 2열 그리드
- 모바일: 1열 그리드
- 특별 고려사항: _______________

== Projects 테이블 DB 구조 정리 ==

프로젝트 정보 (projects 테이블)
- Projects 탭 UI에서 필요한 정보들:
  * id: (프로젝트 고유번호 - 자동으로 1, 2, 3... 증가)
  * title: (프로젝트 제목)
  * description: (프로젝트 설명)
  * tech_stack: (기술 스택 배열 - ["React", "Supabase", "CSS3"])
  * project_type: (개인/팀 여부)
  * detail_url: (배포된 사이트 링크)
  * thumbnail_url: (썸네일 이미지 경로)
  * is_published: (게시 여부 - true/false)
  * sort_order: (표시 순서)
  * created_at: (생성일시)
  * updated_at: (수정일시)
